## ✨ Tentang Projek
**Virtual Photo Studio** adalah aplikasi *web-based photobooth* modern yang dibangun menggunakan **Vue.js**. Terinspirasi dari pengalaman klasik "Webcam Toy", aplikasi ini memungkinkan pengguna untuk mengambil foto secara instan dengan berbagai pilihan *aesthetic frames* yang minimalis dan lucu.

Projek ini dirancang untuk memberikan pengalaman "selfie studio" pribadi langsung dari browser dengan antarmuka yang bersih dan responsif.

## 🌟 Fitur Utama
- **Real-time Webcam Capture:** Pengambilan foto instan dengan latensi rendah.
- **Aesthetic Frame Gallery:** Berbagai pilihan bingkai foto dengan tema *pastel* dan *minimalist*.
- **Instant Download:** Simpan hasil foto langsung ke perangkat dalam format berkualitas tinggi.
- **Responsive Design:** Tampilan yang nyaman digunakan baik di desktop maupun perangkat mobile.
- **Soft UI/UX:** Antarmuka pengguna yang tenang dengan palet warna yang nyaman di mata.

## 🛠️ Tech Stack
- **Frontend Framework:** [Vue.js](https://vuejs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Icons:** Lucide Icons / Font Awesome
- **Styling:** CSS3 (Modern Flexbox & Grid)
